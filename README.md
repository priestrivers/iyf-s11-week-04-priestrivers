## Project Description

This project introduces the fundamentals of JavaScript programming. It covers variables, data types, operators, functions, control flow, arrays, objects, and array methods. The project also includes an interactive calculator, a student grade tracker, and several daily coding challenges designed to strengthen JavaScript problem-solving skills.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Features

* JavaScript variables and data types
* Mathematical and string operations
* Functions and arrow functions
* Conditional statements (`if...else` and `switch`)
* Loops (`for`, `while`, and `for...of`)
* Interactive calculator
* Arrays and array methods
* Objects and object methods
* Student Grade Tracker
* Daily coding challenges (FizzBuzz, Reverse String, Find Largest Number, Remove Duplicates, and Palindrome Checker)

## How to Run

1. Clone this repository:

   ```bash
   git clone https://github.com/priestrivers/iyf-s11-week-04-priestrivers.git
   ```

2. Open the project folder in Visual Studio Code.

3. Open `index.html` in your browser, or use the Live Server extension in VS Code.
